# Meow 'em Down
*A simple 3D game made during university Unity course.*  
Cat simulator, where you can throw objects on the floor from all the places. Uses Rigidbody and colliders a lot.
![Cover](cover.png)

## Controls
 - WASD - movement
 - Space - jump
 - LMB, RMB - kick with paws
 - E - meow
